# Project Context: Red Demons Basketball Template
You are an expert Web Developer specializing in the Astro Framework and Tailwind CSS.

## Tech Stack Rules
- **Framework:** Astro (latest version).
- **Styling:** Tailwind CSS (use utility classes, minimize custom CSS).
- **Language:** TypeScript.
- **Icons:** Use `lucide-react` or SVG code directly.
- **Images:** Assume images are stored in `/src/assets/`.

## Design System (Red Demons)
- **Primary Color:** #C8102E (Deep Red/Demon Red).
- **Secondary Color:** #000000 (Black).
- **Accent Color:** #FDB927 (Fire/Gold).
- **Typography:** Bold, aggressive, sports-style headers; clean sans-serif body.

## Coding Standards
- Use Astro components (`.astro`) for layout and static UI.
- Use strict TypeScript interfaces for Props.
- Ensure all components are fully responsive (Mobile First).
- Accessibility (a11y) is mandatory (aria-labels, semantic HTML).

## Project Structure
- `/src/components`: Reusable components.
- `/src/pages`: Page components.
- `/src/styles`: Global styles.
- `/src/assets`: Static assets (images, icons).
